=== WPMS Maintenance Mode ===

This is a fork of WPMS Site Maintenance mode found at; https://wordpress.org/plugins/wpms-site-maintenance-mode/

Provides an interface to make a WPMS network unavailable to everyone during maintenance, except the admin.

== Description ==

Plugin allows a super admin to put his/her entire network, main site, or subsites into maintenance mode.

== Installation ==

1. Extract wpms-maintenance.zip to the /wp-content/plugins/ directory.
2. Visit Network Admin > Plugins and activate the plugin Network wide.
3. Visit Network Admin > Settings > WPMS Maintenance and choose the site operational option and edit the maintenance message to your liking.
